# 🚀 GitHub Pages 部署指南

## 第一步：在 GitHub 上启用 GitHub Pages

1. 打开浏览器，访问你的仓库：
   **https://github.com/doyourbestt/hangzhou-exhibition**

2. 点击 **Settings**（设置）选项卡

3. 滚动到 **Pages** 部分（在左侧菜单的最下方）

4. 在 **Source** 下：
   - 点击下拉菜单，选择 **Deploy from a branch**
   - 点击第二个下拉菜单，选择 **main** 分支
   - 点击第三个下拉菜单，选择 **/ (root)**
   - 点击 **Save**（保存）按钮

5. 等待 1-2 分钟，页面会刷新

---

## 第二步：查看你的网站

部署完成后，你的网站将出现在 **GitHub Pages** 部分的上方，显示为：

```
Your site is published at https://doyourbestt.github.io/hangzhou-exhibition/
```

---

## 🔄 如何更新网站

以后每次推送代码后，网站会自动更新：

```bash
git add .
git commit -m "更新内容"
git push
```

等待约 2 分钟，网站就会自动更新。

---

## ✅ 完成后

你的网站地址将是：
**https://doyourbestt.github.io/hangzhou-exhibition/**

把这个链接分享给你的朋友，他们就能访问《艺起逛杭州》点赞榜了！

## ⚠️ 注意事项

- 首次部署可能需要 2-5 分钟
- 如果 10 分钟后仍然无法访问，请检查 Settings → Pages 配置
- 确保仓库是 **Public**（公开的），否则 GitHub Pages 无法访问
